# 🎬 YouTube Subscribe Button Animation

A clean and interactive **YouTube-style Subscribe Button** built using pure **HTML & CSS**.
This project focuses on **UI design, smooth animations, and user interaction feedback**.

---

## 🚀 Project Overview

This mini project recreates the **YouTube Subscribe button experience** with:

* ✨ Smooth animation on interaction
* 🎨 Clean and modern UI design
* ⚡ Pure HTML & CSS (no JavaScript)
* 🎯 Focus on micro-interactions

It’s part of a collection of **CSS animation experiments** aimed at improving frontend design skills.
